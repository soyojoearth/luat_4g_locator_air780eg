### 本套代码已经把网络通信交互、LBS定位、内置GPS定位基本完成。外设数据通信还需靠你们自己去做，比如低功耗管理、电源、电池、充电管理、陀螺仪等等其它功能，需要你们自己做上去。

你们只要将本套代码稍稍改一下就可以轻松做出一款4G定位器产品，然后将PCBA量产卖给你们的客户。
